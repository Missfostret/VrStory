Detailed Description

High-level controller that connects a parsed dialogue file (TextAsset) to a UI view (DialogueView) and optional input actions.

Responsibilities:

  • Parse a dialogue file into nodes/steps and create a MiniDialogueRuntime.
  • Subscribe to runtime events (OnLine, OnOptions, OnEnd, OnVariableSet).
  • Drive the UI: type lines, show option buttons, and handle end-of-dialogue state.
  • Handle player input: advance (continue) and option selection.
