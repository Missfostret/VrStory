Detailed Description

Unity bootstrapper that loads a dialogue script from a TextAsset, creates a MiniDialogueRuntime, and wires player input actions to drive the dialogue.

Responsibilities:

  • Parse dialogue nodes/lines from dialogueFile.
  • Instantiate and configure the dialogue runtime.
  • Subscribe to runtime events to display output (currently via Debug.Log).
  • Enable/disable input actions and map them to Continue/Choose calls.

Notes:

Input actions are expected to come from the "Dialogue" action map.

This component starts the dialogue at node "Intro".
