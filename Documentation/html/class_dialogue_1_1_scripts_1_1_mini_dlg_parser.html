Detailed Description

Parser for a lightweight dialogue scripting format that produces a set of named MiniNode objects.

The input text is interpreted as a sequence of nodes, each with a title: header and a body. Nodes are stored in a dictionary keyed by node name (case-insensitive).

Node structure

@subsection autotoc_md0 title: Intro
Hello there!
-> Option A [if hasKey]
<<jump NextNode>>
-> Option B
<<set hasKey = true>>
<<if hasKey>>
<<jump HasKeyNode>>
<<else>>
<<jump NoKeyNode>>
@section autotoc_md1 <<endif>>

Body rules

  • Plain lines become LineStep.
  • Lines starting with -> begin a choice block and become a ChoiceStep.
  • Command lines are wrapped in << ... >> and become CommandStep.
  • <<if ...>> begins a multi-line conditional block ending at <<endif>>.
