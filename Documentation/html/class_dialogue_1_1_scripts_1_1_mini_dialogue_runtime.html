Detailed Description

Executes parsed dialogue nodes and emits events for lines, options, variables, and end-of-dialogue.

The runtime processes a MiniNode step-by-step: LineStep triggers OnLine, ChoiceStep triggers OnOptions, and CommandStep executes an ICommand immediately.

This class is UI-agnostic: it only raises events and exposes methods like Continue and Choose. A view/controller (e.g., DialogueUIController) is expected to subscribe and drive presentation and input.

Variables

Variables are stored in a case-insensitive dictionary and can be substituted into lines using Interpolate. Supported pattern: {varName}.
