Detailed Description

UI controller for displaying dialogue text and presenting selectable dialogue options.

Features:

  • Shows dialogue lines instantly or using a typewriter effect.
  • Spawns option buttons from a prefab and raises an event when clicked.
  • Temporarily disables interaction for one frame to avoid accidental selection/clicks on spawn.

Typical usage:

Call TypeLine or SetLineInstant to show text, and ShowOptions to display choices. Subscribe to OnOptionClicked to be notified when the player selects an option.
