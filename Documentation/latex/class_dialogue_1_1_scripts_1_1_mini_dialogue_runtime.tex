\doxysubsection{Detailed Description}
Executes parsed dialogue nodes and emits events for lines, options, variables, and end-\/of-\/dialogue. 

The runtime processes a \doxylink{class_mini_node}{Mini\+Node} step-\/by-\/step\+: \doxylink{class_line_step}{Line\+Step} triggers \doxylink{class_dialogue_1_1_scripts_1_1_mini_dialogue_runtime_aa4722a044c0aae24e707236022d40981}{On\+Line}, \doxylink{class_choice_step}{Choice\+Step} triggers \doxylink{class_dialogue_1_1_scripts_1_1_mini_dialogue_runtime_a24f65e488b7b2ba448bee8e17964deff}{On\+Options}, and \doxylink{class_command_step}{Command\+Step} executes an \doxylink{interface_i_command}{ICommand} immediately. 

This class is UI-\/agnostic\+: it only raises events and exposes methods like \doxylink{class_dialogue_1_1_scripts_1_1_mini_dialogue_runtime_a4f00ecde4a58e42b44bb983ff779fac8}{Continue} and \doxylink{class_dialogue_1_1_scripts_1_1_mini_dialogue_runtime_ac175155ab9b1d46ee486d9efdf01092f}{Choose}. A view/controller (e.\+g., {\ttfamily \doxylink{class_dialogue_1_1_scripts_1_1_dialogue_u_i_controller}{Dialogue\+UIController}}) is expected to subscribe and drive presentation and input. 

{\bfseries{Variables}}

Variables are stored in a case-\/insensitive dictionary and can be substituted into lines using \doxylink{class_dialogue_1_1_scripts_1_1_mini_dialogue_runtime_aa2b5e89e9a0b89cbde80b9b6ae8e2849}{Interpolate}. Supported pattern\+: {\ttfamily \{var\+Name\}}.
