\doxysubsection{Detailed Description}
Unity bootstrapper that loads a dialogue script from a Text\+Asset, creates a {\ttfamily \doxylink{class_dialogue_1_1_scripts_1_1_mini_dialogue_runtime}{Mini\+Dialogue\+Runtime}}, and wires player input actions to drive the dialogue. 

Responsibilities\+: 
\begin{DoxyItemize}
\item Parse dialogue nodes/lines from \doxylink{class_dialogue_1_1_scripts_1_1_dialogue_bootstrap_aa5243114e16ecfa059b2e642f12def34}{dialogue\+File}. 
\item Instantiate and configure the dialogue runtime. 
\item Subscribe to runtime events to display output (currently via Debug.\+Log). 
\item Enable/disable input actions and map them to Continue/\+Choose calls. 
\end{DoxyItemize}Notes\+: 

Input actions are expected to come from the "{}\+Dialogue"{} action map.

This component starts the dialogue at node "{}\+Intro"{}.
