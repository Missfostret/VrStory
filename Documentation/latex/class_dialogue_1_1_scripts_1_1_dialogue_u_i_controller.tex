\doxysubsection{Detailed Description}
High-\/level controller that connects a parsed dialogue file (Text\+Asset) to a UI view (\doxylink{class_dialogue_1_1_scripts_1_1_dialogue_view}{Dialogue\+View}) and optional input actions. 

Responsibilities\+: 
\begin{DoxyItemize}
\item Parse a dialogue file into nodes/steps and create a {\ttfamily \doxylink{class_dialogue_1_1_scripts_1_1_mini_dialogue_runtime}{Mini\+Dialogue\+Runtime}}. 
\item Subscribe to runtime events ({\ttfamily On\+Line}, {\ttfamily On\+Options}, {\ttfamily On\+End}, {\ttfamily On\+Variable\+Set}). 
\item Drive the UI\+: type lines, show option buttons, and handle end-\/of-\/dialogue state. 
\item Handle player input\+: advance (continue) and option selection. 
\end{DoxyItemize}
