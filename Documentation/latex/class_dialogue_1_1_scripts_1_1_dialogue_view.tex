\doxysubsection{Detailed Description}
UI controller for displaying dialogue text and presenting selectable dialogue options. 

Features\+: 
\begin{DoxyItemize}
\item Shows dialogue lines instantly or using a typewriter effect. 
\item Spawns option buttons from a prefab and raises an event when clicked. 
\item Temporarily disables interaction for one frame to avoid accidental selection/clicks on spawn. 
\end{DoxyItemize}Typical usage\+: 

Call \doxylink{class_dialogue_1_1_scripts_1_1_dialogue_view_a7d71fc70ee3f71527a913d46da3b3e87}{Type\+Line} or \doxylink{class_dialogue_1_1_scripts_1_1_dialogue_view_a3759a183ebf724aadc3d6c1227ec1c10}{Set\+Line\+Instant} to show text, and \doxylink{class_dialogue_1_1_scripts_1_1_dialogue_view_abbd7c24b6170c269bdcb887740bcade4}{Show\+Options} to display choices. Subscribe to \doxylink{class_dialogue_1_1_scripts_1_1_dialogue_view_af81101298f37d9f07b4169d09f6a052a}{On\+Option\+Clicked} to be notified when the player selects an option.
